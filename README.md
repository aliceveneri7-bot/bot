1️⃣ INSTALLAZIONE
----------------
pip install -r requirements.txt

2️⃣ INSERISCI IL TOKEN
-----------------------
Apri app.py e sostituisci:
TOKEN = "INSERISCI_IL_TUO_TOKEN"

3️⃣ AVVIA IL BOT
----------------
python app.py

4️⃣ COMANDI
-----------
/entra          → registra un utente (quando entra nel gruppo)
/lista          → lista utenti e scadenze (solo admin)
/rinnova @nome  → rinnova abbonamento manualmente
